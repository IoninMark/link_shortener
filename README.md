### Сокращатель ссылок:

Тестовый проект с API для получения коротких ссылок и перехода по ним.
Цель - протестировать работу с Fast API и Pydentic.

## Технологии

В проекте применены следующие технологии:

* Python
* FastAPI
* Pydentic

## Запуск проекта

Для запуска необходимо склонировать проект к себе на компьютер:

```
git clone git@github.com:IoninMark/link_shortener.git
```

Далее необходимо в директории создать виртуальное окружение (используется python 3.12)

С помощью uv:
```
uv venv --python 3.12
```
С помощью pip:
```
python3.12 -m venv .venv
```

Активация виртуального окружения:
```
source .venv/bin/activate
```
Установка зависимостей

С помощью uv:
```
uv sync
```
С помощью pip:
```
pip install .
```
Для запуска необходимо в корневой дирректории проекта создать файл .env.
В файле должны быть указаны следующие параметры (на данный момент проект работает с aiosqlite):

```
DATABASE_URL="sqlite+aiosqlite:///./links.db"
BASE_URL="http://localhost:8000"
```

Далее следует выполнить следующие команды для запуска:

```
uvicorn app.main:app --reload
```

# Основные эндпоинты API

Основной эндпоинт (/)

* GET http://127.0.0.1:8000/ — приветственный эндпоинт

Ссылки (/links/)

* POST http://127.0.0.1:8000/links/ — получение новой короткой ссылки
* GET http://127.0.0.1:8000/links/{slug}/ — редирект по короткой ссылке

# Примеры запросов

Создание новой ссылки:
```
POST http://127.0.0.1:8000/links/

{
  "url": "https://example.com/"
}

Response:

{
  "short_url": "http://localhost:8000/AqpsRV",
  "original_url": "https://example.com/"
}
```
